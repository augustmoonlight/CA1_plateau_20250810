# CA1_plateau_20250810
The CA1 model able to produce dendritic NMDA/Ca plateau upon theta burst and ramp frequency co-stimulation per parameters on 2025/08/10.

This is the model associated with the preprint: https://www.biorxiv.org/content/10.1101/2025.10.01.676159v1. 

1. Install required Python packages
```
pip install neuron matplotlib plotly numpy random pickle
```

2. Open **3pathway_minimal_20250810.ipynb** and run the simulation

3. The simulation results and the neuronal morphology with synapses labeled should be as in 
   https://drive.google.com/drive/folders/10OwsaBWBDSF3kuN-pHzu4M_E-du1wL6F?usp=sharing
